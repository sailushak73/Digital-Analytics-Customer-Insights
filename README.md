# Digital-Analytics-Customer-Insights
Python-based digital analytics project focused on customer churn analysis, EDA, and actionable insights.

OverView : 
            This project focuses on analyzing customer data to understand churn behavior
            and derive actionable business insights using digital analytics techniques.

Key Highlights:

• Data cleaning and preprocessing
• Exploratory Data Analysis (EDA)
• Visualization of customer behavior and trends
• Basic machine learning model to support analytical insights

Tech Stack:
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
